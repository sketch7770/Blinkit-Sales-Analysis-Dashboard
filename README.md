Blinkit Sales Analysis Dashboard

Project Overview:
This project focuses on analyzing Blinkit grocery sales data to uncover business insights related to sales performance, outlet establishment trends, item categories, customer ratings, and outlet locations.

The dashboard was built using Power BI after performing data cleaning and preprocessing using Python and Excel.

Objectives:
Analyze total sales performance
Identify top-performing product categories
Compare outlet sizes and locations
Study customer ratings and item visibility
Generate business insights through interactive dashboards

Tools & Technologies Used:
Python (Pandas, NumPy ,matplotlib)
SQL
MS Excel
Power BI
Dataset Information

The dataset contains retail sales information including:
Item Type
Fat Content
Outlet Size
Outlet Location Type
Sales
Ratings
Item Visibility
Establishment Year
Data Cleaning & Preprocessing

Performed the following preprocessing steps:
Removed duplicate records
Handled missing/null values
Standardized categorical values
Corrected inconsistent data entries
Cleaned and formatted numerical columns

Tools used:
Python
Excel
Dashboard Features

The Power BI dashboard includes:
Total Sales Analysis
Average Sales & Ratings
Outlet Performance Comparison
Sales by Item Type
Outlet Size Distribution
Interactive Filters & Slicers
KPI Cards for Business Metrics
Key Insights
Tier 3 outlets generated higher sales compared to Tier 1 and Tier 2 locations.
Snack Foods and Fruits & Vegetables contributed significantly to total sales.
Medium-sized outlets showed strong overall performance.
Certain item categories had high visibility but lower sales conversion.
